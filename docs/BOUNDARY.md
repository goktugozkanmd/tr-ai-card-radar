# Boundary

`tr-ai-card-radar` is a metadata and card review helper.

It can help answer:

- Is a license field visible in Hub metadata?
- Does the card text mention a different license string?
- Is language metadata visible?
- Is access gated?
- Does a model card mention base model, intended use, and limitations?
- Does a dataset card mention source data or creation process?

It cannot answer:

- Is this resource legally reusable?
- Is this resource clinically validated?
- Is this model safe?
- Is this dataset free of copyright problems?
- Is this repository endorsed by its authors for a specific downstream use?
- Which model is better?

Warnings are review prompts, not final judgments.
